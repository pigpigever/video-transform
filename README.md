# Transmute

Desktop video conversion UI scaffold built with Tauri, SolidJS, Kobalte, Tailwind CSS v4, and TanStack Table.

The current skeleton includes:

- A desktop-style queue dashboard for conversion jobs
- Preset, container, audio, and acceleration controls
- A TanStack-powered job table with mock queue data
- Kobalte tabs and progress components for the operations shell
- An FFmpeg command preview area ready to connect to Tauri commands
